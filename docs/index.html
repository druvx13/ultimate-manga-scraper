Ultimate Web Novel & Manga Scraper

Automate your manga and web novel content aggregation with this powerful WordPress plugin

Get Started View on GitHub
Version 2.0.3

Key Features

Multi-Source Scraping

Built-in support for FanFox, WuxiaWorld, NovLove, and any Madara-based manga/novel sites.

Headless Browser Support

Render JavaScript-heavy sites using PhantomJS, Puppeteer, or HeadlessBrowserAPI.

Auto Translation

Translate content on-the-fly using Google Translate, DeepL, or Microsoft Translator.

Cloudflare Bypass

Built-in mechanisms to handle anti-bot protection and Cloudflare challenges.

Automated Scheduling

Cron-based automation for continuous content updates without manual intervention.

Image Management

Download, resize, and store chapter images locally or on cloud storage (S3, Imgur, Flickr).

Proxy Support

Rotate proxies to bypass IP bans and rate limits on target sites.

Madara Integration

Seamlessly integrates with Madara theme's custom post types and storage system.

Quick Start

1

Install Plugin

Upload and activate the plugin in your WordPress admin panel with Madara theme active.

2

Configure Settings

Navigate to Main Settings and enable the plugin. Configure API keys if using translation.

3

Create Rules

Add scraping rules with target URLs, schedules, and processing options.

4

Start Scraping

Run rules manually or let cron automation handle continuous content updates.
